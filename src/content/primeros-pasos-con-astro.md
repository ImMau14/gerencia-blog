## ¿Por qué Astro?

Astro se ha vuelto una de las herramientas más interesantes para construir sitios web estáticos y dinámicos con un enfoque minimalista en JavaScript.

En este artículo veremos cómo configurar un proyecto desde cero, entender la estructura de carpetas y crear nuestra primera página.

![Logo de Astro](https://astro.build/assets/press/astro-logo-light.svg)

### Crear el proyecto

```bash
npm create astro@latest
```

Sigue las instrucciones y elige una plantilla básica. En pocos segundos tendrás un sitio funcional. La filosofía de Astro es enviar cero JavaScript al cliente por defecto, hidratando solo los componentes interactivos que realmente lo necesitan.

¿Ya lo probaste? Cuéntame tu experiencia.
